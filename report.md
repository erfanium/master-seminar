# پروژه خوشه‌بندی بیماران با ناهنجاری ژنتیکی

  

در این پروژه، خط لوله ای طراحی شده است که اطلاعات بیماران را به صورت فایل های VCF در ورودی دریافت می‌کند و پس از مراحلی، چندین خروجی ارائه می‌دهند.

  

## خروجی ها

  

- ماتریکس kinship تولید شده توسط الگوریتم ibs

- خروجی الگوریتم کاهش بعد (PCA)

- لیست مهم ترین جهش های تاثیر گذار بر هر بعد PCA

- خروجی الگوریتم خوشه‌بندی HDBSCAN روی نتایج PCA

- لیست مهم ترین جهش های تاثیر گذار بر هر خوشه

  

## فیلتر های اعمال شده

### مساوی بودن مقدار AF با MAF:

این فیلتر، جهش هایی که بیش از نیمی از بیماران رخ داده اند را نادیده می‌گیرد. این فیلتر حدود **هفت درصد** از جهش های کل دیتاست را نادیده خواهد گرفت.
